# koishi-plugin-adapter-github 文档

## 本地开发

```bash
git clone https://github.com/koishi-shangxue-plugins/koishi-plugin-adapter-github.git
cd koishi-plugin-adapter-github
git checkout docs
yarn install
yarn dev
yarn build
```

## 文档结构

- `/docs` - 文档源文件
- `/docs/.vitepress` - VitePress 配置
- `/docs/markdown` - 各个文档页面
